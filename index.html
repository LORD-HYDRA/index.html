Intro

java is an object oriented internet programming language, it's have following 2 main features:

  1. Platform Independency
  2. Secuity

    3. Object Orientation:it is a way of thinking point of view of object. Its based on real life concepts.following concepts are used in object oriented programming.

    1.Object:Object can be anything which exists physically in the world and can be recongnized to each other . each object have following 3 main features:

    1. States or Variables
    2. Behavior Or Methods
    3. Identification

    Note:May or may not be possible any two objects have same states or methods but never same Identification.

    2.Classification:It is a technique to used for manage similar types of objects in a single logical unit its called class. Class is always virtual it does not class define by class keyword. Class name always start with Upper case and should be noun type.

    Example:Simple, MyTest, ArrayIndexOutOfBoundsException, etc

    3.Generalization:It is a technique to use for manages similar types of classes. Its implemented by Inheritance in java. Interitance is a process to reuse code of existing class by using extends keywords.

    4.Polymorphism:Poly means many and Morphism means Types. So one task can be implements multiple ways. Task is more important compare to its implementation. Polymorphism are following 2 types:

    1. Static Polymorphism or Compile Time Polymorphism (Method Overloading)
    2. Dynamic Polymorphism or Run Time Polymorphism(Method Overriding)

    5.Encapsulation:To put something into capsule. it is used to wrap data in a logic unit it's called class. Outside display only class and its methods name but internally define methods implementiton
