Histroy

History of Java:Java was originally developed by James Gosling at Sun Microsystem (Which has take over by Oracle)in 1991. Initially it was made of small eletronics Devices like SetTop Box , telebisions, etc....

initially it was called OAK. OAK is a symbol of strength and national tree of many country like USA, France , Germany, etc.. But it's occupied by another company so its name changed as JAVA

Features of Java:

  1. Object oriented:In java everthing is an object. each java class also extended by object class
  2. Platform Independent:Java Create .class file so it can be execute on any platform without any modification. So its write once, run anywhere.
  3. Simple:java is designed to be easy to use
  4. Secure:Java code compile and executed by JITC (just is time compiler)so its fully virus-free
  5. Robust:Java can be hendled exception thosse occur at runtime.
  6. Multithreaded:Java can be achieved multitasking by the help of Threads

Rules for Java:

  1. Java sourse file can be contains only single public class
  2. Java source file name must be similar to public class name
  3. Java main() method should be written within public class
  4. A single source file can be contains multiple non public class
  5. Java is highly case Sensitive
